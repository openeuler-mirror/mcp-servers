#  Perf Process Performance Analysis MCP
##  Function Introduction
It provides CPU performance analysis, flame graph generation, and basic functions for obtaining some hardware information, including 14 basic tools:

* top_collect_tool: A tool to obtain the top N resources-consuming items
* get_process_info_tool: A tool to obtain detailed process information
* process_name_to_pid_tool: A tool to convert process name to PID (Process ID)
* perf_collect_tool: A tool for collecting perf.data (Note: The original "攻击" is suspected to be a typo; it is corrected to "工具" (tool) here based on the context of functional description)
* install_perf_tool: A tool for installing perf
* generate_flamegraph_tool: A tool for generating flame graphs
* get_current_time_tool: A tool to obtain the current system time
* write_text_to_file_tool: A tool to write text to a file
* get_memory_usage_tool: A tool to obtain memory usage
* get_cpu_usage_tool: A tool to obtain CPU usage
* get_disk_usage_tool: A tool to obtain disk usage
* get_directory_files_info_tool: A tool to obtain information under a directory
* get_system_info_tool: A tool to obtain system information
* get_cpu_details_tool: A tool to obtain detailed CPU information
##  Usage Instructions
* Root privileges are required.
* The perf.svg file generated by perf will be stored in the src directory, and identified by time + process PID.
* Start the service by running server.py in the src directory.
* Adjust the service output language to Chinese or English by modifying the language field in config.yaml under the src directory to zh or en.
