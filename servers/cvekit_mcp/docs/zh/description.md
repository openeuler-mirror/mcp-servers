# 介绍

patchflow agent 是一个专注于处理代码仓中CVE补丁的工具，支持Gitee、GitCode等多个代码托管平台，旨在自动化CVE漏洞修复流程，提高补丁处理效率。

## 功能特点

- **CVE解析**：自动解析CVE漏洞信息
- **环境搭建**：克隆Linux和kernel源码
- **补丁分析**：获取引入和修复commit ID，分析修复分支
- **补丁应用**：自动应用补丁到目标分支
- **PR创建**：自动创建Pull Request
- **冲突解决**：使用LLM辅助修复补丁冲突
- **批量回移植**：支持批量检查和回移植提交

## 应用场景

- 安全团队批量处理CVE漏洞
- 开发者需要将上游补丁回移植到特定版本
- 自动化CVE修复流程，减少人工干预

## 支持的平台

- Windows
- Linux
- macOS
