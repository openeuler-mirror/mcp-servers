# RPM Builder MCP Server

一个用于构建RPM包的Model Context Protocol (MCP) 服务器，提供自动化的RPM包构建和管理功能。

## 功能特性

- **RPM包构建**: 支持从源代码自动构建RPM包
- **规格文件管理**: 创建和编辑RPM spec文件  
- **依赖解析**: 自动解析和处理构建依赖
- **多架构支持**: 支持x86_64、aarch64等多种架构
- **构建环境管理**: 管理和配置RPM构建环境
- **包验证**: 构建后的RPM包质量检查和验证
